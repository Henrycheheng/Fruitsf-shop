
# 微信小程序-水果商城  优品四季

#### 介绍
优品四季是一个商城类小程序,主要是卖水果,小程序是利用周末时间写的
目前涉及的功能有很多,购物车,倒计时，分享，生成海报分享朋友圈,拼团下单，优惠券等
